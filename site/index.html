Accueil

Vous pouvez parcourir une longue liste de jeux vidéo que vous pouvez trier. Vous avez aussi accès à une courte liste d’éditeurs.

Ensuite vous pouvez effectuer des recherches dans cette base de données par la page formulaire où vous pouvez effectuer des recherches précises et variées.

Encore une fois dans la page formulaire vous pouvez ajouter des jeux vidéo à la base de données.

Le logo animé du menu est un GIF de ma conception qui a été fait image par image afin d’ajouter une petite touche personnalisée au site.

Le site est responsive design et est complètement utilisable sur smartphone.
